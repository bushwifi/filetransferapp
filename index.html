FileShare Pro

A modern, high-speed, cross-platform file and folder transfer tool with a beautiful GUI, built using Python and Tkinter. It supports encrypted peer-to-peer transfers, real-time progress, speed stats, and chat messaging between devices on the same network.

ðŸ’¡ Usage

Follow these simple steps to start sharing files with FileShare Pro.

1

Launch FileShare Pro

Open FileShare Pro on two or more devices connected to the same local network.

2

Discover Devices

Each device will automatically appear in the "Available Devices" list.

3

Connect

Select a device from the list and click Connect to establish a secure connection.

4

Send Files

Select a file or folder and click Send to initiate the transfer.

5

Monitor Progress

Track transfer progress, speed, and chat with peers in real-time.

ðŸ”’ Security

Your privacy and security are our top priorities.

End-to-End Encryption

All file transfers are encrypted using a password-derived key with Fernet (AES) encryption. This ensures that only the intended recipient can access your files.

Local Network Only

FileShare Pro operates exclusively on your local network. No data ever leaves your private network, keeping your files secure from external threats.

No Central Server

The application uses peer-to-peer connections without relying on a central server, eliminating potential points of failure and surveillance.
